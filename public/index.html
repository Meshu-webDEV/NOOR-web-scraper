A web scraping tool using 'Puppeteer'.
NOOR web-scraper is an in-house tool used to scrape the login captcha image of noor.moe.gov.sa allowing end-users to log-in. Then proceeds to scrape a desired set of data.
This website is just a small part of the system, and is used to simply graphically illustrate the scraping functionality.
